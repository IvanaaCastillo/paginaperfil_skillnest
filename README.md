# Página de Perfil 🎨👤

Proyecto desarrollado como práctica de **HTML y CSS**, recreando una página de perfil similar a una interfaz real de red social o plataforma web.  
El objetivo principal fue aplicar estilos CSS, estructuras con Flexbox y buenas prácticas de maquetación.

---

## 📌 Objetivos del Proyecto

- Recrear una página de perfil utilizando HTML y CSS.
- Aplicar conceptos de:
  - Flexbox
  - Posicionamiento
  - Espaciado y alineación
  - Tipografía
  - Colores y estilos visuales
- Mejorar la organización del código y estructura visual.
- Practicar trabajo colaborativo y resolución de problemas en CSS.

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox

---

## 💡 Aprendizajes

Durante este proyecto se trabajó en:

- Creación de layouts responsivos.
- Uso de contenedores flexibles.
- Diseño visual inspirado en interfaces reales.
- Organización de componentes y estilos CSS.
- Resolución de problemas de alineación y distribución de elementos.
